
# Taskify6

![Made with JavaScript](https://img.shields.io/badge/Made%20with-JavaScript-yellow) 
![TailwindCSS](https://img.shields.io/badge/Styled%20with-TailwindCSS-blue) 
![License: MIT](https://img.shields.io/badge/License-MIT-green)

Taskify6 is a lightweight, student-friendly planner built with **HTML, TailwindCSS, and Vanilla JavaScript**.
It helps you organize tasks day-wise, track streaks, and stay motivated with live quotes, history archivr, and visual progress charts.

---
## ✨ Features

- 📅 **Day-wise Task Management**

Add, edit delete, and mark tasks for each day. Limit of 6 tasks per day keeps planning focused.

- 🌗 **Dark/Light Mode Toggle**

Switch themes instantly with smooth particle background animations.

- ✔ **Streak Counter** 

Tracks your daily consistency. Completing tasks increases your streak automatically.

- 📊 **Progress Analysis**

Pie Chart shows completed vs incompleted, they are stored in a history container with day-wise cards.

- 📃  **Task History**

When all tasks of a day are completed, they are stored in a history container with day-wise cards.

- 🧹 **Clear History Quotes**

Reset your history archive anytime with a single click.

- 💭 **Motivational Quotes**
Typerwriter + backspace effect displays rotating quotes to keep you inspired.

- ⌨ **KeyBoard Shortcut**

Press **Enter** to quickly add tasks.

---



## 🛠 Tech Stack

- **Frontend:** HTML, TailwindCSS
- **Logic:** Vanilla JavaScript
- **Storage:** LocalStorage (presistant across sessions)
- **Libraries:**
    - Chart.js (Progress pie chart)
    - particles.js (background animation)
